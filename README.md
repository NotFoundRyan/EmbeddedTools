# 嵌入式开发工具箱

> 一款专为嵌入式开发设计的桌面应用程序，提供 LCD 点阵生成和串口调试等实用工具。

## 简介

我是一名正在学习嵌入式开发的大学生，开发这个工具的初衷是为了方便自己的学习和开发工作。这个工具箱集成了多个常用的嵌入式开发工具，旨在提高开发效率，简化日常操作。

## 功能特性

### LCD 点阵生成
- 支持自定义分辨率（如 320x480 等）
- 支持图片转点阵：上传图片自动转换为黑白点阵
- 支持文本转点阵：输入中英文文本生成点阵
- 可调节阈值，优化点阵效果
- 多种输出格式：C 数组、二进制、十六进制等
- 实时预览点阵效果

### 串口调试助手
- 自动检测可用串口
- 支持多种波特率、数据位、停止位、校验位配置
- 十六进制/文本模式切换
- 自动发送功能
- 实时数据接收和显示
- 数据记录和保存

### 其他功能
- 窗口置顶功能
- 错误日志记录
- 自动检查更新（从 GitHub 获取最新版本）

## 技术栈

- **前端框架**：原生 HTML + CSS + JavaScript
- **桌面框架**：[Tauri](https://tauri.app/)
- **后端语言**：Rust
- **串口通信**：serialport (Rust)

## 安装

### 从 GitHub Releases 下载

1. 访问 [Releases 页面](https://github.com/NotFoundRyan/EmbeddedTools/releases)
2. 下载最新版本的安装包：`EmbeddedTools_0.1.0_x64-setup.exe`
3. 运行安装包，按照提示完成安装

### 从源码编译

```bash
# 克隆仓库
git clone https://github.com/NotFoundRyan/EmbeddedTools.git
cd EmbeddedTools

# 安装依赖
npm install

# 运行开发环境
npm run tauri dev

# 编译发布版本
npm run tauri build
```

## 使用说明

### LCD 点阵生成

1. 在左侧导航栏选择"LCD 点阵生成"
2. 选择输入模式（图片或文本）
3. 上传图片或输入文本
4. 调整目标分辨率和阈值
5. 点击"生成点阵"按钮
6. 选择输出格式并复制结果

### 串口调试

1. 在左侧导航栏选择"串口调试助手"
2. 从下拉列表中选择串口
3. 配置波特率、数据位、停止位、校验位
4. 点击"打开串口"
5. 在发送区输入数据，点击"发送"
6. 接收区会显示接收到的数据
